# Button

常用的操作按钮。

# 基础用法

使用 type、plain、round 和 circle 来定义按钮的样式。

basic.vue
<template>
  <div>
    <button>按钮一</button>
    <button>按钮二</button>
    <button>按钮三</button>
  </div>
</template>
